## Vertailu operaattorit

|  Operaattori | Nimi                        |
|--------------|-----------------------------|
| >            | suurempi kuin               | 
| <            | pienempi kuin               | 
| >=           | suurempi tai yhtäsuuri kuin | 
| <=           | pienempi tai yhtäpieni kuin | 
| ==           | Yhtäsuuri / sama kuin       |
| !=           | Erisuuri / eri kuin         |

## Lookiset operaattorit

|  Operaattori | Nimi                        |
|--------------|-----------------------------|
| &&           | suurempi kuin               | 
| ||           | pienempi kuin               |
